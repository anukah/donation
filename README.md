# Rebuild Peradeniya - Donation Website

A website to support the restoration of University of Peradeniya, Sri Lanka.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your computer
- [Git](https://git-scm.com/) installed

### Clone the Repository

```bash
git clone https://github.com/anukah/donation.git
cd donation
```

### Run Locally

Start a local development server:

```bash
npx -y serve -l 3000
```

Open your browser and go to: **http://localhost:3000**
